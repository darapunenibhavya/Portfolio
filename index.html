Projects

Eye Disease Detection Website

The Eye Disease Detection Website is a deep learning–based web application designed to assist in early identification of common eye diseases. Users can upload an eye image, and the system processes it using a trained convolutional neural network (CNN) model to predict the presence of conditions such as cataract, glaucoma, and others. The platform provides a seamless and user-friendly interface where the uploaded image is analyzed in real time, and the results are displayed with high accuracy. Additionally, based on the predicted disease, the system provides appropriate precautions and recommendations for the user. The website also allows users to download a detailed report containing the diagnosis and suggested precautions.

Rapido UI Clone Website

The Rapido UI Clone is a web application that replicates the user interface of the popular bike taxi service, Rapido. Designed to give users an intuitive and smooth experience, the website offers a clean and modern layout, closely mimicking the original app’s design.With a focus on the user journey, the interface features prominent elements such as ride booking, location selection, and fare estimates. Users can easily input their pickup and drop-off locations, view nearby available riders, and seamlessly check the cost and ride details. This project aims to provide a functional, responsive front-end experience using HTML, CSS, and ReactJS. While the clone does not feature backend functionality (such as ride booking or payment processing), it effectively demonstrates key design elements, navigation flows, and user interactions found in the original Rapido app. The Rapido UI Clone serves as a great learning tool for understanding modern web development practices, including component-based architecture in React and responsive design techniques.
